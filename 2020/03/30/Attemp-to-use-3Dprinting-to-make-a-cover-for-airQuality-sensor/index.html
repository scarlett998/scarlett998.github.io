Attemp to use 3Dprinting to make a cover for airQuality sensor-1

2020-03-30

With the data can be placed in real-time on ThingSpeak,
I decided to design a cover for the sensor.

Here comes 3 function that I want for the current first draft,

  • Waterproof,for example : when the it is raining outside, the rainfall should not to get inside the box.
  • “Breathable”, so the air can go through the box with the inbuilt fan
  • Can be hung, so the sensor can be easily placed.
